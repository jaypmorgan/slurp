# SluRp

*Statistical Lisp in R*


## Examples

```lisp
(rnorm 10 10 10)

# [1]  14.8692128   3.4932029  18.6337963  15.5319696  -2.9426000 -11.6523209
# [7]  -0.9475137   6.1512033   4.6517305   8.0993424

(print "testing")

# "testing"
# "testing" <- returns "testing" as well

# create vectors with [ ]
(mean [1 2 3 4 5])

# [1] 3
```
